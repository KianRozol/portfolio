My Projects

MabisaTalk

Our offline translation app uses Flutter to provide real-time translation in various dialects without needing an internet connection, ensuring seamless and reliable communication anytime. The framwork that was use was Flutter and the programming languages used was Dart

The translation contains two parts: the single input translation and conversational phrases mode. The user will choose if they want to translate a single input or conversational words by pressing the button.

The Mabisatalk App has a learning module feature where the user can learn common phrases, numbers, greetings, and animals. This feature also shows the user how to pronounce the word by pressing the text-to-speech button.

The user can press the text-to-speech button to hear the pronunciation of the translated word.
